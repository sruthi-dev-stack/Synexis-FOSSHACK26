#  Contributing to VeritasData

Thank you for your interest in contributing to VeritasData!



##  How to Contribute

1. Fork the repository
2. Create a new branch  
   Example: `feature/data-cleaning`
3. Make your changes
4. Commit your changes
5. Push to your fork
6. Create a Pull Request

##  Contribution Areas

You can contribute in:

- Data Cleaning Modules
- Machine Learning Models
- Frontend UI Improvements
- API Development
- Documentation

##  Code Guidelines

- Write clean and readable code
- Follow consistent naming conventions
- Add comments where necessary
- Test your code before submitting

##  Commit Message Format

Use clear commit messages:

- `Add feature for missing value detection`
- `Fix bug in duplicate detection`

##  Suggestions

If you have ideas:
- Open an issue
- Discuss before implementing


Thank you for contributing 🚀
