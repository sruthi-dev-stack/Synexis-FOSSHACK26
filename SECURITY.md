#  Security Policy — VeritasData


##  Reporting a Vulnerability

If you find a security issue, please report it responsibly.

Contact:
- Email: sriswarnasruthi2007@gmail.com

Do NOT open public issues for security vulnerabilities.


##  Data Privacy
VeritasData ensures:

- No user data is stored permanently
- Uploaded datasets are processed temporarily
- Sensitive information is not shared

##  Sensitive Data Handling

We detect and handle:

- Emails
- Phone numbers
- Credit cards
- Personal identifiers

Users can:

- Mask data
- Remove sensitive columns
- Anonymize datasets

##  Security Best Practices

- Never upload `.env` files
- Keep API keys private
- Use secure connections (HTTPS)

## Compliance

VeritasData supports:

- GDPR
- HIPAA
- ISO Data Standards


Your data safety is our priority 🔐
