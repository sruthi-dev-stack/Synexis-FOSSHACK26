## 📌 Project Description
VeritasData is a beginner-friendly tool that helps users check the quality of datasets before using them for analysis or machine learning.

Many datasets downloaded from the internet contain hidden issues such as missing values, duplicate rows, incorrect data types, and unrealistic values. These problems can lead to wrong analysis results and inaccurate machine learning predictions.

VeritasData automatically scans datasets, identifies common data quality issues, explains them in simple language, and helps users clean and improve their datasets.

---

## ❗ Problem Statement
Students and researchers often use datasets directly from the internet without verifying their quality. These datasets may contain hidden errors such as missing values, duplicates, incorrect formats, or unrealistic values.

Beginners usually do not have the tools or knowledge to detect these problems before using the data. As a result, poor data quality can lead to unreliable analysis and incorrect machine learning results.

---

## ✅ Proposed Solution
VeritasData provides a simple and beginner-friendly platform where users can upload a dataset and automatically analyze its quality.

The system:
- Detects common data issues
- Calculates a dataset health score
- Visualizes problems through charts
- Explains issues in simple language
- Provides automatic data cleaning options
- Demonstrates the importance of clean data by comparing ML model accuracy before and after cleaning

---

## ⚙️ System Workflow

1. Upload dataset (CSV or Excel)
2. Scan dataset for quality issues
3. Generate dataset health score
4. Visualize detected issues
5. Explain problems in simple language
6. Automatically clean the dataset
7. Compare ML accuracy before and after cleaning
8. Download cleaned dataset
